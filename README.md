# allergy_friendly_restaurants
This is a repository for Carol Lott's HCI 584 project that will create a finder for allergy friendly restaurants.
Users can enter a location (zip code only right now)
Users can search using a radius between 5-50 miiles from the zip code
Users can filter optionally by dietary need (gluten free, vegan, or dairy free right now)
